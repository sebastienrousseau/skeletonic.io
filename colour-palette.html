Colour Palette

Skeletonic CSS comes with a basic set of custom properties, sometimes referred to as CSS variables or cascading variables.

Colour Variables

Skeletonic CSS v.1.2.1 provides a set of colour palettes and variables as a way to store information that we want to reuse throughout our stylesheet. These are designed to be harmonious, ensure accessible content, and help create contrast between UI elements.

Colour Styles available

The Skeletonic CSS v.1.2.1 colour variables palette has a smooth range of shadows ranging from brightest to darkest

You will need to import skeletonic-colours.css to display the colours.
